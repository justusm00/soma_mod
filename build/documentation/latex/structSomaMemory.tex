\doxysubsection{Detailed Description}
Helper structure to manage larger amounts of memory for multiple arrays or states

This struct is intended to store the heavy data of beads, RNG\+\_\+\+STATES, permutation arrays etc. The memory can be dynammically appended. It is internal an householder with precached memory.
