SOMA: SOft coarse grained Monte-carlo Acceleration

Introduction

This is the documentation of SOMA it is intended to help to use the program. In addition it provides a documentation of the functionality of the code. So for developer working with SOMA this is a good starting point for anything. If you have nothing developed for SOMA before, please read our Design Rationale page first.

Background

The program performs a single-chain in mean-field simulation. For further details look for the publication: "Single chain in mean field simulations: Quasi-instantaneous field approximation and quantitative comparison with Monte Carlo simulations", Daoulas, Kostas Ch. and MÃ¼ller, Marcus , J.
