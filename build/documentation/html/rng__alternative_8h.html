Detailed Description

Definition of alternative pseudo random numbers generators for SOMA. PCG32 is the default RNG generation engine, if a different one is used, the definitions for the alternative are in this file.

typedef struct RNG_HEAVY RNG_HEAVY

Struct to hold the global arrays of heany data for alternative PRNGs

Each Phase has one of these structs. And in case an alternative RNG is used, this struct contains the memory for the states. If the memory is not used the pointers are initialized to NULL
