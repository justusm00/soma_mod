◆ update_density_fields()

int update_density_fields ( const struct Phase *const  p)

Update the density fields of the system. The field update fist sets all entries to zero. Then performs a loop over all polymers eg. monomers. The field itself is chosen according to the monomer type. Be shure that there is no additional type set otherwise the adressing fails.

Parameters
pPhase configuration to update the densityfields.
Returns
Errorcode
Note
this call requires MPI collectives so call it on every MPI-core.

◆ update_omega_fields()

void update_omega_fields ( const struct Phase *const  p)

Update the omega -fields of the system.

The function requires updated density fields, as a consequence update_density_fields is called. So the MPI collective characteristics are inherited.

Note
In accordance to the hamiltonian specified via the cmd line interface, the correct hamiltonian is selected.
Parameters
pPhase configuration to update the omega-fields.
Note
this call requires MPI collectives so call it on every MPI-core.
