Detailed Description

Struct which contains the random number generators.

Option to select the pseudo random number enerator (possible values="PCG32", "MT" default=‘PCG32’) Defualt is PCG32. For every other alternative RNG the state only stores an offset to the global array of RNGs. See rng_alternative.h for details.
